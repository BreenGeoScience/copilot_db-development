* Introduction

Copilot is a comprehensive CLI-based accounting system designed for managing multiple business entities, project tracking, and financial reporting.

** System Philosophy

- *Keyboard-driven*: Minimal mouse usage, designed for i3 WM workflow
- *Terminal-first*: All operations via CLI, no GUI required
- *Database-centric*: PostgreSQL as single source of truth
- *Flexible reporting*: Generate reports on-demand via CLI

** Entities Managed

*** BGS (Breen GeoScience)
Consulting business with:
- Project management
- Time tracking
- Invoicing
- Subcontractor management
- Accounts payable

* System Components

** Database: copilot_db

PostgreSQL database with schemas:
- =acc= - Core accounting (transactions, categories, bank accounts)
- =bgs= - Project management (projects, tasks, timesheets, invoices)
- =mhb= - Real estate (properties, rent, mortgages)
- =invest= - Investments (future)

** CLI Application: copilot

Python-based command-line interface:
#+begin_src bash
copilot import       # Import bank transactions
copilot allocate     # Categorize transactions
copilot invoice      # Manage invoices
copilot timesheet    # Enter time
copilot report       # Generate reports
copilot ap           # Accounts payable
#+end_src

* Daily Workflows

** Morning Routine

1. Check accounts receivable
2. Review pending transactions
3. Enter yesterday's time
4. Check project status

** Weekly Tasks

1. Import bank statements
2. Categorize all transactions
3. Review AR aging
4. Generate project status reports

** Monthly Tasks

1. Generate invoices for completed work
2. Pay subcontractors
3. Reconcile bank accounts
4. Generate P&L statements by entity
5. Property income/expense reports

** Year-end Tasks

1. Generate tax summary report
2. Verify all transactions categorized
3. Archive completed projects
4. Prepare accountant package

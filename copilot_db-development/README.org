* Overview

Copilot is a CLI-based accounting system designed for multi-entity project management and financial reporting.

** Features
- Multi-entity accounting (BGS consulting, MHB rental properties)
- Project management with time tracking and invoicing
- Accounts payable/receivable tracking
- Bank transaction imports with smart categorization
- Financial reporting (P&L by entity, cash flow, tax reports)
- Investment tracking (planned for 2026)

** Technology Stack
- PostgreSQL database (=copilot_db=)
- Python 3.x CLI application
- Emacs ORG-MODE documentation
- Keyboard-driven workflow (minimal mouse usage)

* Quick Start

#+begin_src bash
# Clone the repository
gh repo clone BreenGeoScience/copilot_db-development
cd copilot_db-development

# Set up environment
cp .env.example .env
# Edit .env with your database credentials

# Install dependencies
pip install -r requirements.txt

# Initialize database
psql -h 192.168.30.180 -U frank -c "CREATE DATABASE copilot_db;"
psql -h 192.168.30.180 -U frank -d copilot_db -f sql/schema/01_core.sql
#+end_src
